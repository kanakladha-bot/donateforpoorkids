# DonateForPoorKids — NGO Website

A modern, responsive single-page charity website for **DonateForPoorKids**, a grassroots NGO based in Jaipur, Rajasthan, India.

## 🚀 How to Run

Simply open `index.html` in any modern web browser — no build step or server required.

```bash
# Option 1: Double-click index.html
# Option 2: Use VS Code Live Server extension
# Option 3: Python quick server
python3 -m http.server 8080
```

## 📋 Sections

1. **Hero** — Full-width gradient with animated particles, headline, stats
2. **About** — Mission, Vision, Why We Exist with glassmorphism cards
3. **Impact** — Animated counters (5200 children, 120K meals, etc.)
4. **Causes** — 4 cause cards with progress bars (Education, Food, Health, Clothing)
5. **Donation** — Bank details with copy buttons, QR placeholder, payment info
6. **Gallery** — 8-item responsive grid with lightbox
7. **Testimonials** — Auto-playing carousel with 5 donor testimonials
8. **CTA Band** — Full-width call to action
9. **Volunteer** — Registration form with validation
10. **Contact** — Contact form + address + Google Maps embed
11. **Footer** — Full info, social links, bank details

## ✨ Features

- ⚡ Zero dependencies (except Google Fonts + Font Awesome CDN)
